Transaction Alert System

The Transaction Alert system is compartmentalized to provide easy integration with the Core banking system by using data abstraction layer for seamless data exchange and web services for 3rd-Party integration. The Messaging Manager layer offers some SOAP/XML based API for inter-/intra-application communications.

It can connect to any RDBMS (Oracle, SQL Server, or DB2) which houses your core banking system. It can optionally integrate with your Active Directory server to provide single sign-on authentication capability for your application administrators instead of using the traditional authentication method.

In-built into the core system are Smart drivers used in communicating with the various messaging gateway options to provide multimode Messaging. It allows for messages to be delivered to multiple devices allowing the highest level of communication acceptance (e.g. email, mobile-SMS, voice-IVR, IM, pagers, etc.).

The supported messaging gateway options that come with the vanilla version of the software are the SMS and Email services. The other messaging channels (Pager, Voice, etc) can be implemented on the platform as add-ons or extensions as they are implemented by mobile network operators.

Fleet Management System

Our Fleet Management System (FleetMANS) is a leading and robust tool that allows you to effectively manage your fleet assets from any location/branch with numerous SMS/email alerts. It Provides a single and centralized electronic database of company fleet, accessible from all the company locations, for storing factual, vital and operational vehicle usage, maintenance records with drivers information management.

The modular development concept adopted in developing the system enables it to integrate seamlessly with your in-house and/or third-party applications.

Integrated into the system is the VTS module (Vehicle Tracking Service) providing sophisticated tracking technology with flexible reporting software.

It all starts at the Vehicle Location Manager (VLM). The VLM that is installed on your vehicle collects and transmits the tracking data via satellite and/or a cellular network, which in turn delivers the data to the FleetTrack hosted application.

Your web access to the FleetTrack application puts your vehicle information at your fingertips and allows for real time informed fleet management decisions by using the unique features provided by FleetTrack.
